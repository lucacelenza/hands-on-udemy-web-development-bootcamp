Functions Problem Set

Try the functions in the Chrome's console.

  • isEven(number) takes in a number as input and returns whether or not the number is even
  • factorial(number) takes in a number as input and returns the factorial of that number (e.g. factorial(3) => 3 * 2 * 1 = 6)
  • kebabToSnake(kebabCasedString) takes in a, hopefully, kebab-cased string as input and returns its snake-cased equivalent
