Array Problem Set

Code the following javascript functions:

  • printReverse(array): takes an array as an argument and prints out the elements in the array in reverse order (don't actually reverse the array itself)
  • isUniform(array): takes an array as an argument and returns true if all elements in the array are identical
  • sumArray(arrayOfNumbers): accepts as input an array of numbers and returns the sum of all numbers in the array
  • max(arrayOfNumbers): accepts an array of numbers and returns the maximum number in the array
